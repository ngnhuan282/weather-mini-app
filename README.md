# 🌤️ Weather Forecast App (BTQT02)

An Android weather forecast application that provides real-time weather information and forecasts based on the user's current location.

---

## 📌 Overview

This project is part of the **Mobile Advanced Coursework (BTQT02)**.
The application focuses on delivering accurate weather data, forecasts, and visual insights using modern Android development practices.

---

## ✨ Features

* 📍 **Current Location Detection**
  Automatically retrieves the user's location to display local weather.

* 🌡️ **Real-time Weather Information**
  Displays:

  * Temperature
  * Humidity
  * Weather conditions (sunny, rainy, cloudy, etc.)
  * Wind speed

* ⏱️ **Hourly & Daily Forecast**
  Provides weather forecasts by hour and up to multiple days.

* 🗺️ **Weather Map Integration**
  Displays weather data (clouds, temperature, precipitation) using Google Maps.

* 🔔 **Weather Alerts**
  Sends notifications when:

  * Severe weather occurs
  * Temperature exceeds a defined threshold

* 🌍 **Unit Conversion Support**
  Switch between Celsius (°C) and Fahrenheit (°F).

---

## 🧩 Technologies Used

* **Java / Kotlin**
* **Android Studio**
* **REST API (Weather API)**
* **Google Maps API**
* **Location Services (GPS)**
* **RecyclerView (for forecast lists)**

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/ngnhuan282/weather-mini-app.git
```

2. Open the project in Android Studio

3. Add your API keys:

   * Weather API key
   * Google Maps API key

4. Run the app on emulator or real device
